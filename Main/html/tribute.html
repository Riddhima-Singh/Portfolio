Tribute-Page

Technologies Used: HTML, CSS, JavaScript

The Tribute Page is a simple and elegant web project created to honor and showcase the life, achievements, and legacy of a historical or inspirational figure. It focuses on clean design and structured content to provide a meaningful user experience.

Key Features:

  • Well-structured layout with heading, image, and biography section
  • Responsive design for different screen sizes
  • Use of semantic HTML for better readability and accessibility
  • Custom styling for a visually appealing presentation
  • Simple and effective user interface focused on content

This project highlights my understanding of basic web development, semantic structure, and visual design using pure HTML and CSS.
