Hospital Management

Technologies Used: Python

The Hospital Management System is a command-line based application developed using Python. It is designed to manage hospital data including patient records, staff information, and appointment tracking. The system focuses on basic data storage, retrieval, and updates using file handling and modular programming concepts.

Key Features:

  • Patient record management (add, view, search, delete)
  • Basic file-based storage for persistent data
  • Menu-driven interface for ease of use
  • Staff and doctor information tracking (optional extensions)
  • Good demonstration of Python functions and data structures

This project showcases my ability to build functional backend logic using Python and manage real-world data scenarios through structured code.
