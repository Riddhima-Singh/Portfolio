Disaster Management

Technologies Used: HTML, CSS, JavaScript

The Disaster Management System is a web application designed to assist in disaster preparedness, response, and recovery. It provides real-time updates, emergency contact information, and safety tips for users in affected areas.

Key Features:

  • Interactive dashboard for real-time disaster alerts
  • Information on emergency services and evacuation plans
  • User-friendly layout for quick access to critical resources
  • Responsive design for both desktop and mobile users
  • Potential integration with APIs for live weather and seismic data

This project demonstrates my ability to build useful, responsive web interfaces that serve a real-world purpose in public safety and emergency management.
