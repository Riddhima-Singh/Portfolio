Amazon-Clone

Technologies Used: HTML, CSS, JavaScript

The Amazon Clone is a front-end replica of the Amazon e-commerce platform, built using HTML, CSS, and JavaScript. It simulates the layout and features of the real Amazon website, providing a strong demonstration of UI development and design accuracy.

Key Features:

  • Fully layout resembling the original Amazon interface
  • Navigation bar with dropdowns and search functionality
  • Product grid with cards, images, and pricing details
  • Interactive buttons and hover effects for a dynamic user experience
  • Basic cart icon with item count simulation

This project highlights my ability to clone complex UIs, manage styling consistency, and create interactive elements using web technologies.
